# Landing Page — Solução TI

Landing page em Next.js 14, TypeScript e Tailwind CSS, criada para conversão em WhatsApp.

## Executar

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Analytics

Copie `.env.example` para `.env.local` e informe os IDs do Google Analytics e/ou Google Tag Manager. Todo clique em link do WhatsApp envia o evento `whatsapp_click` para o `dataLayer`.

## Antes de publicar

- Substitua os placeholders da seção de avaliações por depoimentos reais do Google.
- Confirme endereço, políticas de garantia e formas de pagamento.
- Configure domínio, GA/GTM e conversão do Google Ads.
